Regression

Regression is essentially finding the line of best fit for a set of data points

Types of Regression:

  • Simple Linear Regression
  • Multiple Linear Regression
  • Polynomial Linear Regression

Regression works well when:

  • there is a relationship between the features and the target
  • you are dealing with continuous numerical data as opposed to categorical data

Regression is pretty good but the results are easily swayed by:

  • outliers
  • relevant features --> can use a correlation coefficient to determine how strong the relationship between the target and the feature is. It'll help determine whether or not to include it.

Measuring machine learning performance can be tricky.

There are a lot of metrics that we use to evaulate the machine learning result. The ones we choose depends on what machine learning method we have used and the context of the data.

Some examples are:

  • Accuracy
  • Precision and Recall
  • F1 Score
  • Log-Loss
  • K-fold Validation

Accuracy = correct predictions / total predictions * 100%

Accuracy with Imbalanced Classes

Lets imagine that your model always no matter what thinks that every picture is a picture of a cat.

Now in your testing set it turns out that 99/100 of your test cases are actually cats and there is one dog.

That means your accuracy will be 99%. But if you had a testing set that had only 1/100 cats, then you would have an accuracy of 1%

This problem can be fixed by:

  • Balancing the test cases
  • increase the minorities
  • Synthesise without exact replicas to avoid overfitting
  • Decrease majorities
  • TP = true positive, FP = false positive, FN = false negative
  • precision: TP/(TP + FP): The amount that were actually out of what you said were cats
  • recall: TP/(TP + FN) : the amount that were actually cats out of all the cats

Use this when you have a small data set and it isnt wise to split into training and testing sets

  • Section off the data into K sections. Each section should be a reflection of all the classes
  • pick one out of the K sections, this one is for testing, all the others are for training
  • Train and then test. Get your results and keep them
  • Choose a new section for testing a repeat

By the end you should have K lots of results which can be collated to have a substantial amount of results to be analysed

Making each section a reflection of all the classes is called stratification. You're just trying to make sure each test has a all the classes so we dont get the problem with accuracy and class imbalances.
