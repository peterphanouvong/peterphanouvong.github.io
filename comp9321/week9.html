Clustering

Clustering is a form of unserpervised learning. It is similar to classification but there are no labels.

Unsupervised Learning

  • no labels
  • Capture the structure of interest
  • Visualise the structure of a dataset
  • Discover groups and outliers
  • Can be used to extract features for supervised learning

There are many types of clustering:

  • Flat: into groups
  • Hierarchal: into hierarchy
  • Hard: Data points belong to only one group
  • Soft: Data point can belong to more than one group

The Basic Ideas

The basic idea is to group the similar points together in a cluster

But the question is how do we determine which point is in which cluster?

  • Pick K arbitrary points
  • take the distance between these points to all the other points
  • A data point will be assigned a cluster if it is the closest cluster

Properties of K-Means

  • Guarenteed to converge in a finite number of itereations
  • Running time per iteration will differ
  • Assign data points to the closest centre takes O(kN) time

K-Means Variations/parameters

  • Specifying the inital centres changes what the final clustering will look like
  • There is a strategy to get better clusters
    • Pick the points to be spaced out maximally
  • Maxing out the spacing between points is not always the best

Good clustering:

  • High IntraClass similarity (Compactness): data points within the same cluster are similar
  • Low InterClass similarity (Isolation): data points between clusters are not similar at all

Limitations of K-Means

  • Works only on simple clusters (SMILEY FACE)
  • data points have to be blobbed together to be detected as a cluster
  • Complex clusters will go undetected

Hierarchal Clustering

type of unserpervised machine learning

Two types:

  • Agglomerative: combining clusters into one big cluster
  • Divisive: start with a mega cluster that gets broken down into smaller ones

Linkage Criteria

This refers to the distance between clusters. How do we measure it?

  • Single Linkage : smallest distance between two data points from different clusters
  • Complete Linkage : greatest distance between two data points from different clusters
  • Average Linkage : distance between the centroids of the clusters

Each linkage criteria will result in a different hierarchy

Proximity Matrix

When forming the hierarchy, a proximity matrix is made, mapping the distance between clusters

The closest clusters are then merged together to form a new cluster

The process is repeated until there is only one cluster

How do we choose how many clusters? (Divisive)

Dendrogram

A dendrogram is a tree diagram

  • When 2 clusters are joined, join them in the dendrogram
  • The height of the join is equal to the distance between the clusters
  • Set the threshold distance such that it cuts throught the middle of the tallest vertical line
  • the number of clusters = the number of lines intersected by the threshold

Advantages

  • Easy to implement
  • No need to decide the number of clusters

Disadvantages

  • Not suitable for a large dataset
  • sensitive to outliers
  • intial seeds have a big impact on the final result
  • linkage criteria is selected rather arbitrarily
